Selenium Automation: Test Automation for private commercial website

Selenium Test Automation to test an actively developing commercial product:

I have built a test suite of 15 GUI tests and growing. The product is a modern website for tracking and monitoring user data. There are different access levels, functionality, and data under test.

This test suite is integrated with Git Actions, and has both triggered and automatic test suite run. I've used open-source tools on Git to enable easily-viewed test results.
